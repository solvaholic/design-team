# Example Project

This is a template project structure. Copy this folder to create a new design project.

## Structure

```
example_project/
├── currentstate.json       # Central index of project state
├── insights/              # Research, observations, and synthesis
├── ideas/                 # Solution ideas and their evolution
└── playbacks/            # Presentation materials and decisions
```

## Getting Started

1. Copy this folder and rename it to your project name
2. Update `currentstate.json` with your project details
3. Use the templates in `/templates` to create your research and design artifacts
4. Keep `currentstate.json` updated as you progress through design phases

## Design Phases

- **Empathize:** Understand users and their context
- **Define:** Synthesize insights and define the problem
- **Ideate:** Generate solution ideas
- **Prototype:** Build to validate ideas
- **Iterate:** Validate with users and iterate
